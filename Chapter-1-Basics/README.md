## Introduction

### Data

Data are characteristics, usually numerical that are collected through observation. In a more technical context, Data are a set of values of qualitative or quantitative variables about one or more persons or objects. In short, any facts and figures about an entity is called as Data.

### Information

Processed Data is called Information. Data is transformed into information when they are viewed in context or in post-analysis

Say for example, the following table is the table of data, as nothing could be understood due to lack of context.

|     |     |     |
| --- | --- | --- |
| 373 | 40  | 36  |

But the moment we add context to it , it transforms into usable information. Now we can understand that the table is giving us information related to output figures of a motorbike

| Engine Displacement | Power (BHP) | Torque (Nm) |
| ------------------- | ----------- | ----------- |
| 373                 | 40          | 36          |
| 250                 | 24.1        | 21          |

### Database

A database is an organized collection of data generally stored and accessed electronically from a computer system.

# Database Management System

The Database Management System is the software that interacts with end users, applications and the database itself to capture,analyse and modify the data. The DBMS software additionally encompasses the core facilities provided to administer the database. The sum total of the database, the DBMS and the associated applications can be referred to as a Database System
